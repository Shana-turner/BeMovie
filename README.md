# BeMovie 
Projet réalisé en équipe de deux 

Le but du projet est d'utiliser un api. C'est un site qui permet de trouver des films, d'avoir un résumé, d'avoir le nombre d'étoile du film, ... 

### Réalisé en 
- Html
- Scss
- Javascript
- Rest API
